O que vimos?

A aua foi apenas para construir o nosso autorretrato. Assim como, os expressionistas usavam cores e formas para representa os sentimentos. Usamos a mesma técnica só que como um autorretrato

Atividade

Eu comecei desenhando um rosto, com essa base coloquei os olhos em tamanhos diferentes, uma boca exagerada. As cores eu misturei-as no rosto, mas fora as separei, como se cada emoção ficasse no seu canto por fora, mas por dentro em uma bagunça. As formas foi só para representas os sentimentos.
